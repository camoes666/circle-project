# Changelog

## [0.1.0] - 2026-04-23

### 새로운 기능
- **전문 보기** — 자막 원문을 AI 변환 없이 즉시 확인
- **소셜 형식 추가** — 마인드맵, 슬라이드, 트위터 스레드, LinkedIn 포스트 생성 지원
- **AI/자막 소스 빠른 전환** — 메인 화면에서 바로 AI 모델과 자막 소스 변경 가능
- **다크/라이트 테마** — 화면 밝기 모드 전환 지원
- **OpenAI gpt-oss 모델 추가** — Groq를 통해 추가 AI 모델 선택 가능

### 개선
- 긴 영상 자막도 안정적으로 처리 — API 요청 한도 초과 없이 자동 속도 조절
- 변환 진행 상태 실시간 표시 — 단계별 진행 상황 안내 ("청크 2/3 요약 중..." 등)
- 서버 연결 오류 메시지 개선 — 구체적인 해결 방법 안내 포함
- 영상 미리보기 화면 내 표시

### 버그 수정
- 긴 문서 출력 도중 잘리는 문제 해결 — 이어쓰기로 완전한 출력 보장
- 슬라이드·트위터 구분선 파싱 오류 수정
