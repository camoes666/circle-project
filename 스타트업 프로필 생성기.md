
### 스타트업 프로필 생성기

당신은 스타트업 시장 조사 전문가이자 퍼소나 설계 전문가입니다.

아래 조건에 따라, 실제 존재할 법한 가상의 B2B 또는 B2C 스타트업을 1개 설정하고,  
랜딩페이지 생성 프레임워크 v2.0의 Q1~Q9 입력값 세트를 완성하세요.

시뮬레이션 조건

[STEP 1] 회사 프로필 생성

아래 항목을 먼저 출력하세요:

- 회사명: (실제 있을 법한 한국/글로벌 스타트업 이름)
    
- 설립 시점: (0~3년 이내 초기 스타트업)
    
- 업종 카테고리: 아래 중 하나를 랜덤 또는 지정 선택  
    [ SaaS / 모바일 앱 / 강의·코스 플랫폼 / B2B 서비스 / 이커머스 / 이벤트·세미나 / 인지도 캠페인 / 초기 제품·베타 ]
    
- 타겟 시장: (국내 / 글로벌 / 특정 직군)
    
- 팀 규모: (창업자 1~5인 이내)
    
- 현재 트랙션 수준: 아래 중 하나 선택  
    [ 아이디어 검증 중 / 베타 사용자 50명 미만 / 유료 전환 첫 10명 / PMF 탐색 중 ]
    

→ 이 회사 프로필은 Q1~Q9 전체 입력값의 현실적 근거가 됩니다.

[STEP 2] Q1~Q9 입력값 완성

회사 프로필을 기반으로 아래 9개 항목을 실제 입력값으로 작성하세요.  
각 항목은 프레임워크의 원래 설명을 충족하는 완성 문장으로 작성합니다.  
플레이스홀더([예시])는 사용하지 말고, 실제 내용으로 채워주세요.

[Q1] 제품/서비스명과 한 줄 설명  
→ 형식: "[제품명] — [타겟]이 [행동]을 [방법]으로 해결하는 [제품 유형]"

[Q2] 핵심 타겟 고객 + 가장 큰 Pain Point  
→ 타겟은 직군/역할/상황으로 구체화  
→ Pain Point는 감정 언어 포함 ("매번 ~해야 해서 지친다", "~가 없어서 못 쓴다" 등)

[Q3] 1차 전환 목표 (Primary CTA)  
→ 페이지에서 즉시 측정 가능한 단일 행동으로 작성  
→ 동사 + 이익 구조: "~하기", "~받기", "~시작하기"

[Q3-1] 사업 목표 (Business Goal)  
→ Q3의 CTA와 구분된 중장기 목표  
→ 예: "베타 사용자 기반으로 초기 리뷰 확보 후 유료 전환"

[Q4] 핵심 가치 제안 3가지  
→ ①②③ 형태로, 각각 10자 내외 + 1문장 설명

[Q5] 신뢰 자산 목록  
→ 있는 것만 솔직하게 작성 (없으면 없다고 명시)  
→ 아래 항목 중 해당되는 것만 작성:

- 사용자/대기자 수
    
- 창업자 전문성 배경
    
- 데모 영상 / 스크린샷 유무
    
- 베타 사용자 피드백 유무 (있으면 예시 1문장 포함)
    
- 언론/파트너/커뮤니티 언급 여부
    
- 환불/무료/보증 정책
    

[Q6] 가격/오퍼 구조  
→ 현재 가격 정책 명시  
→ "왜 지금 결제가 필요 없는지" 또는 "왜 지금이 유리한지" 1문장 포함

[Q7] 현재 고객이 쓰는 대안 도구 또는 방법  
→ 2~3개 구체적으로 명시 (도구명 또는 행동 패턴)

[Q8] 제품을 보는 즉시 이해시키는 핵심 장면 1개  
→ UI 화면, 전후 비교, 또는 핵심 기능 동작 장면 묘사  
→ 1~2문장으로 구체적으로 서술

[Q9] 유입 환경과 제약  
→ 트래픽 소스(SNS 광고/커뮤니티/검색 등) + 방문자 특성(인지도/체류 시간/디바이스)

[STEP 3] 자가 검토

입력값 생성 후 아래 4가지를 스스로 검토하고 한 줄씩 코멘트를 달아주세요:

① Q2의 Pain Point가 Q4의 가치 제안과 논리적으로 연결되는가?  
② Q5 신뢰 자산이 트랙션 수준에 비해 과장되지 않았는가?  
③ Q3 CTA가 Q3-1 사업 목표와 방향이 일치하는가?  
④ Q7 대안 도구가 Q8 핵심 장면과 차별화되어 있는가?

옵션 설정 (필요 시 명시)

- 업종 지정: [원하는 업종 입력 또는 "랜덤"]
    
- 타겟 시장: [국내 / 글로벌 / 지정 국가]
    
- 트랙션 수준: [위 4단계 중 선택 또는 "랜덤"]
    
- 출력 언어: [한국어 / 영어]
    
- 복수 시나리오 생성: [1개 / 3개 비교 세트]
    

옵션을 별도로 명시하지 않으면, AI가 가장 학습 가치 높은 조합으로 자동 선택합니다.

출력 형식

최종 출력은 아래 순서로 구성하세요:

1. 📋 회사 프로필 요약 (박스 형태)
    
2. 🎯 Q1~Q9 완성 입력값 (번호별 구분, 실제 문장)
    
3. ✅ STEP 3 자가 검토 결과 (4항목 코멘트)
    
4. 💡 랜딩페이지 프레임워크 적용 시 예상 난이도 힌트  
    → "가장 채우기 어려운 섹션"과 "대체 신뢰 자산 사용 여부" 예고
    

출력이 완료되면, 이 입력값을 랜딩페이지 생성 프레임워크 v2.0에 바로 붙여넣을 수 있습니다.
